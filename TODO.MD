# План повышения информационной связности MCP (Connectivity & Density)

## 1. Интеграция с Git (Real-time Status)
- [ ] **Git-статус в инструментах модификации**
    - Внедрить автоматический вызов `git status --porcelain` после операций `edit`, `create`, `move`, `delete`, `rename`.
    - Добавлять информацию о состоянии файла в ответ инструмента (Modified, Added, Deleted, etc.).
    - **Цель**: Понимание состояния репозитория без лишних вызовов `git_cmd`.

## 2. Улучшенный поиск (Contextual Search)
- [ ] **Контекстные строки (`beforeContext` / `afterContext`)**
    - Добавить параметры для захвата строк вокруг найденного совпадения в `SearchFilesTool`.
    - Поддержка вывода вида: `line-1`, `TARGET LINE`, `line+1`.
    - **Цель**: Снижение необходимости вызова `read_file` при анализе результатов поиска.

## 3. Детализация батч-редактирования
- [ ] **Расширенный отчет Multi-file Batching**
    - Возвращать статистику по каждому файлу: количество примененных операций (repl/ins/del) и итоговый CRC32.
    - **Цель**: Посекундная верификация сложных изменений.

## 4. Gradle Intelligence
- [ ] **Smart Error Parser для Gradle**
    - Реализовать фильтрацию логов сборки для выделения ошибок компиляции (File, Line, Message).
    - Выводить `ERROR SUMMARY` в конце ответа `GradleTool`.
    - **Цель**: Быстрая навигация по ошибкам сборки.

## 5. Прозрачность сессии
- [ ] **Сводка AccessTracker в журнале транзакций**
    - Добавить в `TransactionJournalTool` список файлов, доступных на запись в текущей сессии.
    - **Цель**: Самоконтроль прав доступа.
