# ЭТАП: Кросс-инструментальная атомарность (ЗАВЕРШЕНО)
- [x] **Nesting Support в TransactionManager**: Поддержка вложенных вызовов транзакций.
- [x] **Atomic Safety**: Гарантия того, что commit происходит только на самом верхнем уровне.
- [x] **batch_tools**: Инструмент для объединения вызовов разных инструментов в одну транзакцию.
- [x] **Sequential Execution**: Последовательное выполнение цепочки действий.
- [x] **Testing**: Тесты на сложные цепочки и атомарный откат.

# СЛЕДУЮЩИЙ ЭТАП: Java-специфичные инструменты
- [ ] **java_get_outline**: Извлечение структуры класса (импорты, сигнатуры методов/полей).
- [ ] **java_find_usages**: Поиск использований символа (базовая реализация через поиск текста с фильтрацией).
- [ ] **java_check_style**: Проверка соответствия отступов и именования.
