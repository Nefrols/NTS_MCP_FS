# НОВЫЙ ЭТАП: Кросс-инструментальная атомарность

## 1. Рефакторинг транзакций
- [ ] **Nesting Support в TransactionManager**: Поддержка вложенных вызовов транзакций.
- [ ] **Atomic Safety**: Гарантия того, что commit происходит только на самом верхнем уровне.

## 2. Оркестрация
- [ ] **batch_tools**: Инструмент для объединения вызовов разных инструментов в одну транзакцию.
- [ ] **Sequential Execution**: Передача результатов (или хотя бы состояния ФС) между шагами батча.

## 3. Тестирование
- [ ] Тесты на сложные цепочки (Move + Edit + Delete).
- [ ] Тесты на откат всей цепочки при ошибке в любом звене.
